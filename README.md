# shortenURL
Saving space and improved readability when you sharing the URL
## 短網址生產器
使用者可以輸入想要進行縮短的網址，並且在縮短網址後，可以在網域名稱內，以及網頁中直接前往原本的網址。

## install step
 1. clone repo into your computer
 2. cd into {repo path}/shortURL
 3. npm install
 4. npm run dev
